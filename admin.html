How it works

  • Changes are saved to localStorage in your browser.
  • The public site reads localStorage overrides first, then falls back to data.json.
  • Use Export JSON to download your settings as a backup.
  • To make changes permanent across devices, replace data.json with your exported file (via your hosting file manager or re-deploy).
